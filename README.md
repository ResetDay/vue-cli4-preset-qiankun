# 基于vue-cli4的前端脚手架

## 模板类型

- PC端模板
- 移动端模板
- qiankun主应用
- qiankun微应用

## 使用说明

```bash
# 安装 vue-cli4
npm install -g @vue/cli

# 远程 preset 创建项目
vue create --preset ResetDay/qiankun-vue-cli4-preset my-project
# 或者克隆到本地创建
vue create --preset direct:https://github.com/ResetDay/qiankun-vue-cli4-preset.git my-project --clone

# 预览
cd my-project
npm run serve
```
